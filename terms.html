3. App Functionality

3.1 What TopoXY Does

TopoXY is a coordinate conversion and terrain analysis application that:

  • Converts coordinates between Stereo70 (Romanian projection) and WGS84 (GPS)
  • Provides elevation data and terrain analysis
  • Generates elevation profiles between two points
  • Displays ANCPI ortophoto satellite imagery for Romania
  • Analyzes slope, aspect, and terrain features
  • Allows batch coordinate conversion

3.2 Data Sources

TopoXY uses data from various sources including:

  • Mapbox for elevation data and base maps
  • ANCPI (Romanian National Agency for Cadastre) for ortophoto imagery
  • OpenStreetMap for topographic feature data

IMPORTANT: Coordinate conversions and elevation data are provided for informational purposes. For official surveying or legal purposes, always verify results with certified professionals and official sources.
